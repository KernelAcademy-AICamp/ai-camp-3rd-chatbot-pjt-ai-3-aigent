# Copaung Code Command | 키워드 소싱 레이더

네이버 데이터랩 인기 키워드를 시계열로 학습하고 쿠팡 경쟁도를 결합해 **미래 유망 키워드 Top 10**과 **쿠팡 최적화 제품명**을 추천하는 Next.js + Tailwind + Groq + Supabase 프로젝트입니다.

## 주요 기능
- 기간·카테고리 기반 키워드 수집/필터링 (비의류·비부피형)
- 시계열 예측(ARIMA/Prophet/LSTM 등 확장 가능)으로 성장률/계절성 산출
- 쿠팡 경쟁도·가격 지표를 가중치로 반영한 스코어링 및 Top 10 추천
- Groq LLM을 통한 쿠팡 검색 패턴 기반 니치 제품명 제안 (대화형)
- Supabase에 추천 로그 적재 가능 (추후 KPI/대시보드 활용)

## 빠른 시작
1) 환경변수 작성
```bash
cp .env.example .env.local
# 값을 채워주세요
# NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
# NEXT_PUBLIC_SUPABASE_ANON_KEY=...
# GROQ_API_KEY=...
```

2) 로컬 실행
```bash
npm install
npm run dev
# http://localhost:3000
```

3) Supabase 로그 테이블(optional)
```sql
create table if not exists chat_logs (
  id bigint generated by default as identity primary key,
  product_name text,
  target text,
  focus text,
  reply text,
  created_at timestamp with time zone default now()
);
```

## 주요 파일
- `src/app/page.tsx` : 랜딩 + 대화형 제품명 추천 UI
- `src/app/api/chat/route.ts` : Groq LLM 호출 및 Supabase 로그 처리
- `src/lib/groq.ts`, `src/lib/supabase.ts` : 클라이언트 헬퍼
- `.env.example` : 필요한 환경변수 정의

## 배포 가이드
- Vercel에 연결 후 `npm run build` 기반 자동 배포
- Supabase URL/Key, GROQ_API_KEY를 Vercel 환경변수에 등록
- 필요 시 Cron/Edge Function으로 데이터랩 크롤링·예측 파이프라인을 추가하십시오.
